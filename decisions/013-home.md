# Decision #013 — Home

**วันที่:** 2026-07-15
**สถานะ:** Proposed

## บริบท

Home screen เดิม (จากบรีฟแรกสุด) ออกแบบเป็นปุ่ม Start Session ตรงกลาง — ยังไม่มีการนิยามว่า Home ควรทำหน้าที่อะไรกันแน่ในเชิงจิตวิทยาการใช้งาน

## การตัดสินใจ

Home ไม่ใช่ Dashboard แต่คือหน้าจอ "เตรียมตัวออกงาน" (Prepare for Work) ตอบคำถามเดียว: วันนี้พร้อมออกไปทำงานหรือยัง

## เหตุผล

สอดคล้องกับหลัก Less Input, More Driving — คนขับไม่ควรต้องอ่าน/ตัดสินใจอะไรเยอะก่อนออกกะ
